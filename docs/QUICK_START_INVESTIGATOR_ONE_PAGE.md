# TIER™ CaseFlow — Investigator Quick Start *(one page)*

**1. Sign in** → Open your app URL → enter email/password → **Log in**.

**2. Sidebar (main areas)** → **Mission Control** (dashboard) · **Intake Folder** (new intake) · **Communication Hub** (match attorney messages to cases) · **Case Files** (your cases) · **Investigative Tasks** (directives & due dates) · **Workload Log** (calendar of activity) · **Report Hub** (reports) · **Voucher Hub** (billing status) · **System Settings** (profile-linked identity + **Export & backup**: JSON + CSV for *your* cases/tasks).

**3. My Profile** (bottom of sidebar) → Set **Full name**, **Sidebar initials**, **Agency name** → **Save**.

**4. Open a case** → **Case Files** → click a case → **case jacket** opens.

**5. Case jacket tabs** → **details** (case data) · **logs** (add **date**, **code**, **narrative**, **hours**) · **communication** (log calls/emails) · **evidence** (requests & receipt tracking) → save/sync as shown.

**6. Tasks** → **Investigative Tasks** → work by **due date** / **priority** → mark **complete** → fix **needs intake** items per office policy.

**7. Communications** → **Communication Hub** → open item → **match to case** → save.

**8. Vouchers** → **Voucher Hub** → pick segment (e.g. Missing / Submitted / Paid) → open case from list → update **voucher status** in jacket if instructed.

**9. Reports** → **Report Hub** → choose report type (e.g. Weekly, Stagnant, Pre-Trial, **Global Intel Brief**) → set format if offered → **generate** → copy/save output.

**9b. Backup / save your data** → **System Settings** → **Export & backup** → **Download my data (JSON)** (full snapshot for your assigned cases) and/or **Export my case registry** / **Export my tasks** (CSV).

**10. Sign out** → sidebar → **Log out**.

**Need help?** Wrong/missing cases → ask admin to **assign** the case to you. Login issues → admin or IT.
